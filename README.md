# Laravel CRUD Demo - PROJ

Projeto de demonstração de um CRUD funcional utilizando Laravel 12, desenvolvido como parte de um fluxo de trabalho profissional integrando **Jira** e **GitHub**.

## 🚀 Tecnologias
* **PHP:** 8.5
* **Framework:** Laravel 12
* **Banco de Dados:** MySQL
* **Servidor Local:** Laragon / Artisan Serve

## 🛠️ Funcionalidades
- [x] Listagem de produtos (Index)
- [x] Cadastro de novos itens (Create/Store)
- [x] Edição de registros (Edit/Update)
- [x] Exclusão de dados (Delete)

## 🔧 Como rodar o projeto
1. Clone o repositório:
   `git clone git@github.com:seu-usuario/laravel-crud-demo.git`
2. Instale as dependências:
   `composer install`
3. Configure o arquivo `.env` com suas credenciais de banco de dados.
4. Rode as migrations:
   `php artisan migrate`
5. Inicie o servidor:
   `php artisan serve`

## 📊 Gestão de Projeto
Este repositório está vinculado a um board do **Jira Software**. Os commits seguem o padrão de chaves de tarefas (ex: `PROJ-1`).
